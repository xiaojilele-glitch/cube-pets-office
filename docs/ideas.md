# 立方宠物办公室 - 设计构思

## 项目概述

使用 Three.js 构建一个3D交互式场景，展示多智能体LLM系统论文内容。立方宠物在办公室家具场景中"干活"——阅读论文、在电脑前工作、搬运文件、讨论内容。同时集成Codex API配置面板。

## 方案一：像素办公室 (Pixel Office)

**设计运动**: 等距像素风 (Isometric Voxel Aesthetic) — 受 Crossy Road、Monument Valley 启发

**核心原则**:

1. 低多边形等距视角，营造玩具盒般的微缩世界感
2. 鲜明的色块对比，无渐变，纯色面
3. 信息以"便签纸"和"白板"形式嵌入3D空间
4. 交互感如同操控一个活的桌面玩具

**色彩哲学**:

- 主色：暖木色 (#D4A574) 作为地板和桌面基调
- 强调色：薄荷绿 (#7ECFB3) 用于交互元素和高亮
- 辅助色：珊瑚粉 (#FF8A80) 用于宠物标记和通知
- 背景：淡蓝灰 (#E8EDF2) 模拟天花板/天空

**布局范式**:

- 全屏3D场景占据主视口，等距45度俯视角
- 左侧浮动面板显示论文内容（可折叠）
- 右下角AI配置抽屉
- 宠物点击后弹出气泡对话框

**标志性元素**:

1. 立方宠物头顶的"思考气泡"显示论文摘要
2. 迷你白板上渲染论文图表
3. 地面上散落的"论文页"可被宠物捡起

**交互哲学**: 点击宠物触发工作动画，拖拽论文页到白板上展示内容

**动画**:

- 宠物idle时轻微上下弹跳（正弦波）
- 走路时身体左右摇摆
- 工作时头部点头动作
- 场景切换时整体缩放过渡

**字体系统**:

- 标题: "Press Start 2P" 像素字体
- 正文: "Noto Sans SC" 中文无衬线

## 方案三：温馨书房 (Cozy Study Room)

**设计运动**: 斯堪的纳维亚温暖极简 (Scandinavian Warm Minimalism) — 受 Hygge 文化、Studio Ghibli 工作室场景启发

**核心原则**:

1. 温暖的自然光照明，柔和的阴影，营造午后书房的舒适感
2. 家具和物品精心摆放，每个角落都有故事
3. 立方宠物是"研究助手"，各有分工——有的在电脑前打字，有的在搬书，有的在白板前讨论
4. 论文内容自然融入场景——贴在墙上、放在桌上、投影在屏幕上

**色彩哲学**:

- 主色：奶油白 (#FFF8F0) 墙壁和UI背景，传递温暖和开放
- 木质色：蜂蜜棕 (#C4956A) 家具和地板，带来自然质感
- 强调色：深森绿 (#2D5F4A) 植物和交互按钮，象征知识生长
- 点缀色：陶土橙 (#D4845A) 书脊和宠物配饰，增添活力
- 文字色：炭灰 (#3A3A3A) 确保可读性

**布局范式**:

- 3D场景占据上方2/3，自由旋转的书房场景
- 下方1/3为论文阅读面板，可上拉全屏
- 右侧边栏为AI助手配置（齿轮图标触发）
- 场景中的物品可点击，触发论文相关内容展示

**标志性元素**:

1. 墙上的软木板钉着论文关键图表，宠物会走过去指着讨论
2. 桌上的台灯发出温暖的圆锥形光，照亮工作区域
3. 窗外有缓慢飘动的云朵，暗示时间流逝

**交互哲学**:

- 观察者视角，像透过窗户看一间忙碌的研究室
- 点击任何家具或宠物都有响应
- 鼠标悬停时物品轻微发光提示可交互
- 滚轮控制场景缩放，拖拽旋转视角

**动画**:

- 宠物走路时的Q弹摇摆，每步都有微小的压缩和回弹
- 打字宠物的手指快速敲击键盘动画
- 搬书宠物缓慢但稳定地在书架和桌子间往返
- 讨论宠物面对面，头部交替转动
- 台灯光线微微摇曳，模拟真实灯泡
- 窗帘随"微风"轻轻飘动
- 场景加载时从模糊到清晰的景深过渡

**字体系统**:

- 标题: "Playfair Display" 优雅衬线体，传递学术气质
- 正文: "Noto Sans SC" 清晰的中文无衬线体
- 代码/配置: "JetBrains Mono" 等宽字体

## 选择：方案三 - 温馨书房 (Cozy Study Room)

选择理由：

1. 与Kenney家具套件的风格最为契合——低多边形的温暖家具天然适合斯堪的纳维亚风格
2. 立方宠物作为"研究助手"的设定最有趣味性和叙事感
3. 论文内容可以最自然地融入场景（墙上的图表、桌上的文件、屏幕上的内容）
4. 温暖的色调和柔和的光照能让用户长时间停留而不感到疲劳
5. 交互设计直觉性强，不需要学习成本
