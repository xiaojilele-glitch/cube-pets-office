# 实现计划：向量数据库与 RAG 管道

## 概述

按模块分组实现 RAG 管道，从共享类型和配置开始，逐层构建摄入→分块→嵌入→索引→检索→增强→反馈→生命周期→可观测性→前端展示的完整链路。每个模块实现后紧跟测试，确保增量验证。
