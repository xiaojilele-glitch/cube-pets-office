# 实施计划：workspace-visual-unification

## 概述

本 spec 负责把当前分裂的页面风格收口成一套统一的工作台视觉语言，优先治理低频页和明显割裂页。

## Worktree 并行建议

- 建议独立 worktree owner 负责样式 token、共享页壳和低频页治理
- 在任务中台布局稳定前，尽量少碰 `TasksPage.tsx`
- `LineagePage.tsx` 可作为低风险先手页面单独推进
