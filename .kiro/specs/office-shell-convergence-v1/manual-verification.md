# 办公室主壳收敛方案 v1 手工验证

## 1. 唯一入口验证

1. 打开首页 `/`
2. 观察首屏是否明确告诉用户这里就是任务入口
3. 验证用户无需跳去 `/tasks` 或 `/command-center` 才能开始任务

## 2. 路由兼容验证

1. 访问 `/command-center`
2. 验证是否跳回首页或只出现极轻兼容提示
3. 访问 `/command-center/legacy`
4. 验证不再出现旧指挥中心完整界面

## 3. 深链验证

1. 访问 `/tasks/:taskId`
2. 验证任务详情仍可查看
3. 访问 `/replay/:missionId`
4. 验证回放仍可使用

## 4. 导航验证

1. 打开桌面导航和移动端导航
2. 验证首页是唯一高频入口
3. 验证低频能力不再占据主导航

## 5. 文案与心智验证

1. 检查首页、任务页、兼容页文案
2. 验证不会同时出现“办公室 / 任务台 / 指挥中心都是主入口”的冲突表达
